# 抽卡機在哪！Card Radar — 規格文件

**網站網址：** https://cardradartw.vercel.app/  
**GitHub Repo：** https://github.com/romiajoin/taiwan-gacha-map  
**最後更新：** 2026/06/03

---

## 專案概述

社群共建的台灣抽卡機 / IP 快閃活動查詢網站，資料由管理者維護於 Google Sheet，網站自動讀取並顯示。

---

## 技術架構

| 項目 | 工具 |
|------|------|
| 前端 | 純 HTML / CSS / JavaScript（單一 index.html） |
| 地圖套件 | Leaflet.js 1.9.4（OpenStreetMap 底圖，免費） |
| 資料來源 | Google Sheet（發布為公開 CSV） |
| 圖片託管 | Cloudinary |
| 網站託管 | Vercel（免費） |
| 字體 | Noto Sans TC、Space Mono（Google Fonts） |

**不需要後端、不需要資料庫、不需要 API 金鑰。**

---

## Google Sheet 欄位規格

| 欄位 | 欄位代號 | 說明 | 必填 |
|------|----------|------|------|
| 店名 | A | 活動或地點名稱 | ✅ |
| 場地 | B | 所在建築或商場（如：三創生活 7F） | ❌ |
| 縣市 | C | 縣市名稱（如：台北市） | ❌ |
| 地址 | D | 詳細地址 | ✅ |
| 緯度 | E | 數字，用於地圖定位 | ✅ |
| 經度 | F | 數字，用於地圖定位 | ✅ |
| IP | G | 熱門角色或 IP 名稱（如：芙莉蓮） | ❌ |
| 期間限定 | H | 活動日期（如：2026/05/29～2026/07/23） | ❌ |
| 備註 | I | 補充說明 | ❌ |
| 圖片 | J | Cloudinary 圖片網址 | ❌ |

> 緯度經度可以用 Google Maps 點地點後取得。  
> 欄位為空時，對應資訊不顯示，不影響版面。

## 功能規格

### 地圖
- 以台灣為中心（23.6N, 121.0E），預設縮放層級 8
- 地標圖示：🎰 emoji
- 點地標 → 跳出彈窗
- 手機版彈窗有 `max-height` 限制（`calc(100vh - 320px)`）並支援捲動，避免被上方裁切

### 彈窗（點地標後）
顯示以下資訊（有資料才顯示）：
1. 圖片（點擊可在新分頁開啟完整圖片）
2. 店名（黃色標題）
3. 🏢 場地
4. 📍 地址
5. ⭐ IP
6. 📝 備註
7. ⏰ 期間限定（黃色圓角框）
8. 在 Google Maps 查看 →（粉紅色連結，點擊用地址開啟 Google Maps）

### 側邊欄列表
每個地點卡片顯示：
1. 店名
2. ⏰ 期間限定（有才顯示，黃色）
3. Tags：IP、縣市（粉紅色圓角 tag）

點卡片 → 地圖飛到該地點並打開彈窗

### 搜尋
- 搜尋框 placeholder：「搜尋 IP 角色」
- 可搜尋：店名、地址、縣市、IP 角色
- 即時過濾，不需按 Enter

### 右下角統計
顯示目前地點總數

### 回報表單
- Google 表單，含兩個分頁：新增地點 / 資訊有誤
- 選擇回報類型後跳到對應分頁，填完直接提交
- 表單連結：https://forms.gle/1yDKadx89DoesrSj7
- 回應資料寫入同一份 Google Sheet 的獨立分頁，由管理者審核後手動更新

## RWD（響應式設計）

| 裝置 | 版面 |
|------|------|
| 桌機（>768px） | 左側邊欄 + 右側地圖並排 |
| 手機（≤768px） | 上方列表（230px）+ 下方地圖（剩餘空間） |

### 手機版列表樣式
- 列表底色：`#1c1c1e`
- 搜尋框：膠囊形（border-radius: 19px）、半透明灰底
- 卡片背景：`rgba(120,125,128,0.2)` 半透明
- Tag：深底白字白邊框

## 資料更新流程

1. 管理者在 Google Sheet 新增/編輯資料
2. 網站重新整理後自動讀取最新資料（不需要任何操作）

### 新增圖片流程
1. 上傳圖片到 Cloudinary
2. 複製圖片網址
3. 貼到 Google Sheet J 欄
